## 🧭 Navegación rápida

* `Cmd + O` → Buscar archivo por nombre
* `Cmd + Shift + O` → Buscar símbolo (funciones, structs, etc.)
* `Cmd + E` → Archivos recientes
* `Cmd + Shift + E` → Ubicaciones recientes
* `Cmd + Click` → Ir a definición
* `Cmd + B` → Ir a declaración
* `Cmd + Alt + B` → Ir a implementación
* `Ctrl + H` → Jerarquía de tipos
* `Cmd + U` → Ir a super método / trait

---

## 🔎 Búsqueda avanzada

* `Cmd + F` → Buscar en archivo
* `Cmd + R` → Reemplazar en archivo
* `Cmd + Shift + F` → Buscar en todo el proyecto
* `Cmd + Shift + R` → Reemplazar en todo el proyecto

---

## ✍️ Edición de código

* `Cmd + D` → Duplicar línea
* `Cmd + Delete` → Borrar línea
* `Alt + Shift + ↑ / ↓` → Mover línea
* `Cmd + /` → Comentar línea
* `Cmd + Alt + /` → Comentar bloque
* `Cmd + Shift + Enter` → Completar statement automáticamente
* `Ctrl + Space` → Autocompletado básico
* `Ctrl + Shift + Space` → Autocompletado inteligente

---

## ⚡ Refactor & code actions

* `Shift + F6` → Rename (variable, función, etc.)
* `Cmd + Alt + M` → Extraer método
* `Cmd + Alt + V` → Extraer variable
* `Cmd + Alt + C` → Extraer constante
* `Cmd + Alt + P` → Extraer parámetro
* `Ctrl + T` → Surround with (if, match, etc.)
* `Cmd + Shift + A` → Buscar acción (clave total 🔥)

---

## 📦 Imports (MUY importante)

* `Option + Enter` → Auto-import / fix rápido
* `Ctrl + Option + O` → Optimizar imports (elimina no usados + ordena)
* `Cmd + Alt + L` → Reformat code (también ordena cosas)
* `Cmd + Shift + I` → Ver definición rápida (sin salir)

💡 Tip: Si ves un error tipo *"cannot find function"*, apretá `Option + Enter` y RustRover te sugiere el import automáticamente.

---

## 🧹 Limpieza & formato

* `Cmd + Alt + L` → Formatear código
* `Ctrl + Option + O` → Limpiar imports
* `Cmd + Alt + Shift + L` → Formateo avanzado (configurable)
* `Cmd + Alt + S` → Settings (para rustfmt, clippy, etc.)

---

## 🧪 Testing

* `Ctrl + Shift + R` → Run test
* `Ctrl + Shift + D` → Debug test
* Click derecho → Run/Debug
* `Cmd + Shift + F10` → Run contexto actual

---

## 🐞 Debugging

* `Cmd + F8` → Toggle breakpoint
* `Ctrl + D` → Debug
* `F8` → Step over
* `F7` → Step into
* `Shift + F8` → Step out
* `Cmd + Alt + R` → Resume program

---

## 🪟 Manejo de ventanas

* `Cmd + 1` → Project panel
* `Cmd + 4` → Run panel
* `Cmd + 5` → Debug panel
* `Cmd + W` → Cerrar tab
* `Cmd + Shift + ] / [` → Cambiar entre tabs

---

## 🔀 Multicursor & selección

* `Option + Click` → Cursor múltiple
* `Ctrl + G` → Seleccionar siguiente ocurrencia
* `Cmd + Ctrl + G` → Seleccionar todas
* `Alt + Drag` → Selección en bloque

## 🧠 Productividad PRO

* `Double Shift` → Buscar TODO (archivos, acciones, clases 🔥)
* `Cmd + Shift + A` → Ejecutar cualquier acción
* `Cmd + P` → Ver parámetros de función
* `Cmd + Q` → Quick documentation
* `Ctrl + Space (2x)` → Autocomplete más agresivo

---

## 🦀 Específico de Rust

* `Option + Enter` sobre error → Fix automático (imports, traits, etc.)
* Sugerencias de traits → Auto-implement con intención
* Quick fix para `unwrap`, `match`, `if let`, etc.

---

# 🧩 Tips clave (de los que te hacen volar)

* Usá **`Option + Enter` TODO el tiempo** → es el cerebro del IDE
* **`Cmd + Shift + A`** si no sabés un shortcut
* Limpiá imports seguido (`Ctrl + Option + O`)
* Formateá siempre (`Cmd + Alt + L`)
* Navegá con `Cmd + Click` en vez de buscar manualmente
